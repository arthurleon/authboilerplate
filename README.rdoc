==MailCatcher

Utilizando o MailCatcher no ambiente de desenvolvimento.

Esta pequena gem é um servidor SMTP simples que captura os emails enviados para porta 1025 (smtp://localhost:1025).

NÃO DEVE SER INSTALADO VIA GEMFILE!!!

<b>Instalação</b>

<tt>$ gem install mailcatcher

$ mailcatcher &</tt>

Para ver os emails acesse:

http://localhost:1080/
